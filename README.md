# ICT2104_STISTraight

<h2>How to Setup</h2>
<p>1. Install APK from Andriod APK folder</p>
<p>2. Setup HC-05, 2x MPU6050 and Vibration motor onto MSP432</p>
<p>3. Connect phone to HC-05, pairing password is 1234</p>
<p>4. Good to go!</p>
